Advanced Tech Quiz

Test your knowledge with 15 questions

1. Which HTML element is used for a self-contained composition like a blog post?
Correct Answer: <article>
Explanation: The <article> element represents a self-contained composition, such as a blog post, forum post, or news article.
2. The HTML <picture> element allows responsive images based on device width.
Correct Answer: True
Explanation: The <picture> element allows you to specify different images for different screen sizes or resolutions, enabling responsive images.
3. Which CSS unit is relative to the font-size of the root element?
Correct Answer: rem
Explanation: The rem unit is relative to the font-size of the root element, making it useful for consistent scaling across a page.
4. CSS Grid allows for two-dimensional layout designs.
Correct Answer: True
Explanation: CSS Grid is a two-dimensional layout system, allowing both rows and columns to be controlled.
5. Which JavaScript keyword declares a block-scoped variable?
Correct Answer: let
Explanation: The `let` keyword declares a block-scoped variable, unlike `var` which is function-scoped.
6. The CSS pseudo-class :nth-child() selects elements based on their position in the parent.
Correct Answer: True
Explanation: The :nth-child() pseudo-class selects elements based on their position within their parent element.
7. Which HTML element defines a container for SVG graphics?
Correct Answer: <svg>
Explanation: The <svg> element is used as a container for SVG graphics, allowing vector-based drawings in HTML.
8. The 'position: sticky' property allows an element to stick to the viewport when scrolling.
Correct Answer: True
Explanation: `position: sticky` allows an element to stick to a specified position within the viewport until a boundary is reached.
9. Which HTML element is used to embed a JavaScript file?
Correct Answer: <script>
Explanation: The <script> tag is used to embed or reference JavaScript code in HTML.
10. The localStorage API stores key-value pairs in the browser.
Correct Answer: True
Explanation: localStorage stores key-value pairs persistently in the browser, even after the page is closed.
11. Which CSS property creates smooth transitions between property changes?
Correct Answer: transition
Explanation: The CSS `transition` property allows smooth animation effects when CSS properties change.
12. Arrow functions in JavaScript bind their own 'this' context.
Correct Answer: False
Explanation: Arrow functions do not bind their own `this`; they inherit `this` from the surrounding context.
13. Which CSS property controls the visibility of overflow content?
Correct Answer: overflow
Explanation: The `overflow` property controls what happens when content exceeds the container's box.
14. The CSS flexbox property 'justify-content' aligns items horizontally.
Correct Answer: True
Explanation: The flexbox property `justify-content` aligns items along the main axis, which is horizontal by default.
15. Which HTML element is used to create a clickable button?
Correct Answer: <button>
Explanation: The <button> element creates a clickable button that can trigger actions in forms or scripts.
