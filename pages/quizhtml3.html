HTML Advanced Quiz

Test your advanced knowledge of HTML elements

1. Which HTML element is used to define a self-contained composition, like a blog post?
Correct Answer: <article>
2. True or False: The HTML <picture> element allows responsive images based on device width.
Correct Answer: True
3. Which JavaScript method is used to parse a string to an integer?
Correct Answer: parseInt
4. Which CSS unit is relative to the font-size of the root element?
Correct Answer: rem
5. True or False: CSS Grid allows for two-dimensional layout designs.
Correct Answer: True
6. Which HTML attribute is used to provide a tooltip when hovering over an element?
Correct Answer: title
7. Which JavaScript keyword declares a block-scoped variable?
Correct Answer: let
8. True or False: The CSS pseudo-class :nth-child() selects elements based on their position in the parent.
Correct Answer: True
9. What HTML5 API allows you to store key-value pairs in the browser?
Correct Answer: localStorage
10. Which CSS property is used to create smooth transitions between property changes?
Correct Answer: transition
11. Which HTML element defines a container for SVG graphics?
Correct Answer: <svg>
12. True or False: In JavaScript, arrow functions bind their own 'this' context.
Correct Answer: False
13. Which CSS property allows you to control the visibility of overflow content?
Correct Answer: overflow
14. Which HTML element is used for embedding external JavaScript files?
Correct Answer: <script>
15. True or False: The 'position: sticky' property allows an element to stick to the viewport when scrolling.
Correct Answer: True
