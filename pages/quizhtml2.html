HTML Intermediate Quiz

Test your knowledge of HTML fundamentals

1. What is the main purpose of the HTML <section> element?
Correct Answer: To group related content semantically

Explanation: The <section> element is used to group related content that shares a common theme. It gives semantic meaning to the page structure, helping search engines and screen readers understand the content.
2. True or False: The <div> element has semantic meaning in HTML.
Correct Answer: False

Explanation: The <div> element is non-semantic. It does not describe the meaning of its content and is mainly used for layout and styling purposes.
3. Which HTML attribute improves accessibility by providing text for screen readers?
Correct Answer: alt

Explanation: The alt attribute provides alternative text for images. Screen readers use this text to describe images to visually impaired users, improving accessibility and SEO.
4. Which CSS property is used to change text color?
Correct Answer: color

Explanation: The color property is used in CSS to change the text color. Other options like font-color and text-style are not valid CSS properties.
5. True or False: The CSS box model includes margin, border, padding, and content.
Correct Answer: True

Explanation: The CSS box model consists of four parts: content, padding, border, and margin. These parts define the size and spacing of elements on a webpage.
6. Which CSS property is used to create space inside an element?
Correct Answer: padding

Explanation: The padding property creates space inside an element, between the content and its border.
7. Which HTML element is best for embedding a YouTube video?
Correct Answer: <iframe>

Explanation: YouTube videos are embedded using an <iframe>. It allows external content to be displayed inside a webpage securely.
8. True or False: CSS Flexbox is one-dimensional.
Correct Answer: True

Explanation: CSS Flexbox is one-dimensional because it works in either a row or a column. For two-dimensional layouts, CSS Grid is more suitable.
9. Which HTML tag is used to define navigation links?
Correct Answer: nav

Explanation: The <nav> element is a semantic HTML tag used to define navigation links on a webpage.
10. Which CSS property controls the stacking order of elements?
Correct Answer: z-index

Explanation: The z-index property controls the vertical stacking order of elements. Elements with a higher z-index appear in front of elements with a lower value.
